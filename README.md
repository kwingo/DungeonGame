# Dungeon Game (C#)

A simple console-based dungeon game built in C# to practice object-oriented programming and core programming concepts.

## 🧠 Overview

This project was created to strengthen my understanding of C# fundamentals, including classes, methods, and control flow. The game simulates a basic dungeon environment where the player navigates and interacts with game elements.

## 🚀 Features

- Console-based gameplay
- Object-oriented design (classes and methods)
- Basic game logic and player interaction
- Control flow handling for game progression

## 🛠️ Technologies Used

- C#
- .NET (Console Application)

## 📚 What I Learned

- Fundamentals of C# syntax and structure
- Object-oriented programming concepts
- Writing clean and organized code
- Problem-solving through game logic implementation

## 🔄 Future Improvements

- Add more complex game mechanics (health, enemies, inventory)
- Improve user input handling
- Refactor code for scalability
- Add a graphical interface (optional)
